# web-capture

## 基于 

* ffmpeg-3.4.8
* emscripten-2.0

## 支持编码

* H.264
* H.265
* Mpeg2
* Mpeg4
* VP8
* VP9

## 支持格式

* Mkv
* Mov
* Avi
* Mp4
* Webm

## 注意事项

* 编译前需要下载 [ffmpeg-3.4.8.tar.xz](http://ffmpeg.org/releases/ffmpeg-3.4.8.tar.xz) 并解压至与 `web-capture` 同级的目录

* `emsdk` 安装目录需要与 `web-capture` 同级
